# Projet de Gestion des Ressources Humaines

Ce dépôt contient un modèle Power BI (.pbit) pour la gestion des ressources humaines.

## Champs inclus

- EmployeeID
- EmployeeName
- Salary
- Position
- State
- DateOfBirth
- Gender
- MaritalStatus
- HiringDate
- TerminationDate
- EmploymentStatus
- Department
- RecruitmentSource
- PerformanceScore
- EngagementSurvey
- EmployeeSatisfaction
